# ✊✋✌ Rock Paper Scissors (Streamlit)

A fun Rock Paper Scissors game built using Python and Streamlit where users play against the computer.

## 🚀 Features

- Play against the computer
- Random computer choices
- Win/Lose/Draw detection
- Score tracking
- Simple Streamlit interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Random

## 📂 Project Structure

```
Rock_Paper_Scissors_Streamlit/
│
└── rps.py
```

## ▶️ How to Run

Install Streamlit

```bash
pip install streamlit
```

Run

```bash
streamlit run rps.py
```
